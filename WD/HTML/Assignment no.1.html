Assignment No.1: Study of Web Technology Basics & Languages Overview

1. What is Web Technology?

Web technology refers to the tools and languages used to create and run websites and web applications. It includes both frontend and backend technologies.

2. Frontend Technologies

  • HTML - Structures the web content.
  • CSS - Styles the HTML content.
  • JavaScript - Adds interactivity to web pages.

3. Backend Technologies

  • Node.js - JavaScript runtime for server-side programming.
  • Python (Django, Flask) - Easy-to-learn and powerful backend options.
  • PHP - Widely used server-side scripting language.
  • Go (Golang) - Fast and efficient backend development.
  • Java (Spring) - Used in enterprise-level web apps.

4. Database Technologies

  • MySQL - Popular SQL database.
  • PostgreSQL - Advanced open-source SQL database.
  • MongoDB - NoSQL database for flexible data storage.

5. Web Servers

  • Apache - Open-source and widely used web server.
  • Nginx - High-performance web server and reverse proxy.
